# API

属性 说明 类型 默认值
type 按钮类型，可选值：[ default | primary | danger ] string default
size 按钮尺寸，可选值：[ large | small ] string -
shape 按钮形状，可选值：[ circle ] string -
loading 按钮 loading 状态 boolean false
long 块级按钮 boolean false
icon 按钮图标，具体值可参见 https://material.io/tools/icons/ string -
